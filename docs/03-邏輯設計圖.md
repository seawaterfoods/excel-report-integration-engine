## 一、主流程 — 報表合併邏輯

```mermaid
flowchart TD
    START(["▶ 開始執行"]) --> LOAD_CFG["載入 application.yml<br/>取得 year / month / 路徑"]
    LOAD_CFG --> RESOLVE_DIR["解析處理目錄<br/>import/{year}/{month}/"]
    RESOLVE_DIR --> CHECK_DIR{"目錄是否存在？"}

    CHECK_DIR -->|否| ERR_DIR["❌ 錯誤：目錄不存在"]
    ERR_DIR --> END_FAIL(["⬛ 結束 (失敗)"])

    CHECK_DIR -->|是| SCAN_FILES["掃描目錄<br/>找出 templet/ 與匯入檔"]
    SCAN_FILES --> FIND_TPL{"找到 templet<br/>樣板檔？"}

    FIND_TPL -->|否| ERR_TPL["❌ 錯誤：缺少樣板檔"]
    ERR_TPL --> END_FAIL

    FIND_TPL -->|是| VALIDATE["前置檢核"]
    VALIDATE --> V_COUNT{"檔案數量 ==<br/>預期匯入總數？"}

    V_COUNT -->|否| ERR_COUNT["❌ 錯誤：數量不符<br/>預期 N 份，實際 M 份"]
    ERR_COUNT --> END_FAIL

    V_COUNT -->|是| V_NAME["檢核所有檔名<br/>前兩碼是否為數字"]
    V_NAME --> V_NAME_OK{"全部通過？"}

    V_NAME_OK -->|否| ERR_NAME["❌ 錯誤：檔名格式不符<br/>列出不合規檔案"]
    ERR_NAME --> END_FAIL

    V_NAME_OK -->|是| COPY_TPL["複製樣板檔<br/>作為產出檔基底"]
    COPY_TPL --> INIT_TRACKER["初始化已寫入<br/>座標追蹤器"]
    INIT_TRACKER --> LOOP_START{"還有未處理<br/>的匯入檔？"}

    LOOP_START -->|否| OUTPUT["產出檔案命名<br/>[日期]_[報表名稱].xlsx"]
    OUTPUT --> SAVE["儲存產出檔至<br/>output/{year}/{month}/"]
    SAVE --> GEN_REPORT["產出執行報告"]
    GEN_REPORT --> END_OK(["✅ 結束 (成功)"])

    LOOP_START -->|是| READ_FILE["讀取下一份匯入檔<br/>第一個工作表"]
    READ_FILE --> EXTRACT["萃取獨有資料<br/>(比對樣板找差異)"]
    EXTRACT --> DUP_CHECK{"座標是否<br/>已被寫入？"}

    DUP_CHECK -->|是| ERR_DUP["❌ 錯誤：重複寫入<br/>紀錄衝突座標與來源檔"]
    ERR_DUP --> END_FAIL

    DUP_CHECK -->|否| WRITE_CELL["寫入產出檔<br/>對應座標"]
    WRITE_CELL --> TRACK["更新座標追蹤器"]
    TRACK --> LOOP_START

    style START fill:#4caf50,color:#fff
    style END_OK fill:#4caf50,color:#fff
    style END_FAIL fill:#f44336,color:#fff
    style ERR_DIR fill:#ffcdd2
    style ERR_TPL fill:#ffcdd2
    style ERR_COUNT fill:#ffcdd2
    style ERR_NAME fill:#ffcdd2
    style ERR_DUP fill:#ffcdd2
```

## 二、資料萃取與比對邏輯

```mermaid
flowchart TD
    START(["讀取單一匯入檔"]) --> OPEN["開啟匯入檔<br/>取得 Sheet[0]"]
    OPEN --> LOAD_TPL["載入樣板檔<br/>所有 cell 座標與值"]

    LOAD_TPL --> ITER["遍歷匯入檔<br/>所有非空 cell"]
    ITER --> CELL{"取得 cell<br/>(座標, 值)"}

    CELL --> IN_TPL{"該座標在<br/>樣板中存在<br/>且值相同？"}

    IN_TPL -->|是<br/>共用資料| SKIP["跳過<br/>(不需寫入)"]
    SKIP --> HAS_NEXT{"還有下一個<br/>cell？"}

    IN_TPL -->|否<br/>獨有資料| COLLECT["加入待寫入<br/>清單"]
    COLLECT --> HAS_NEXT

    HAS_NEXT -->|是| CELL
    HAS_NEXT -->|否| RETURN(["回傳待寫入清單<br/>List&lt;CellData&gt;"])

    style START fill:#2196f3,color:#fff
    style RETURN fill:#2196f3,color:#fff
```

## 三、重複座標檢核邏輯

```mermaid
flowchart TD
    START(["接收待寫入清單"]) --> ITER["遍歷清單中<br/>每個 CellData"]
    ITER --> CELL{"取得<br/>(座標, 值, 來源檔)"}

    CELL --> EXISTS{"座標已在<br/>追蹤器中？"}

    EXISTS -->|否| ADD["加入追蹤器<br/>Map&lt;座標, 來源檔&gt;"]
    ADD --> NEXT{"還有下一個？"}

    EXISTS -->|是| CONFLICT["記錄衝突<br/>原寫入檔 vs 目前檔"]
    CONFLICT --> ABORT(["❌ 中止處理<br/>輸出衝突報告"])

    NEXT -->|是| CELL
    NEXT -->|否| OK(["✅ 檢核通過<br/>可安全寫入"])

    style START fill:#ff9800,color:#fff
    style OK fill:#4caf50,color:#fff
    style ABORT fill:#f44336,color:#fff
```

## 四、年月篩選邏輯

```mermaid
flowchart TD
    START(["讀取 year / month 設定"]) --> HAS_YEAR{"year<br/>有設定？"}

    HAS_YEAR -->|是| HAS_MONTH_Y{"month<br/>有設定？"}
    HAS_YEAR -->|否| HAS_MONTH_N{"month<br/>有設定？"}

    HAS_MONTH_Y -->|是| CASE1["處理 import/{year}/{month}/<br/>僅該年該月"]
    HAS_MONTH_Y -->|否| CASE2["處理 import/{year}/*/<br/>該年所有月份"]

    HAS_MONTH_N -->|是| CASE3["處理 import/*/{month}/<br/>每年該月"]
    HAS_MONTH_N -->|否| CASE4["處理 import/*/*/<br/>所有年月"]

    CASE1 --> EXEC(["進入主流程"])
    CASE2 --> EXEC
    CASE3 --> EXEC
    CASE4 --> EXEC

    style START fill:#9c27b0,color:#fff
    style EXEC fill:#4caf50,color:#fff
```

## 五、產出檔命名邏輯

## 六、Class 關係圖

```mermaid
classDiagram
    class AppConfig {
        -String importDir
        -String outputDir
        -String processYear
        -String processMonth
        -String processReports
        +getProcessReportList() List~String~
    }

    class ScopeConfig {
        -ReadMode readMode
        -String startCell
        -String endCell
        -Integer expectedFileCount
    }

    class ReportOrchestrator {
        -FileService fileService
        -ExcelService excelService
        -ValidationService validationService
        -MergeService mergeService
        +run(args) void
        +execute() ProcessResult
        -processReport(task) ReportResult
        -outputReport(result) void
    }

    class FileService {
        -AppConfig appConfig
        +resolveReportTasks() List~ReportTask~
        +findTemplateFile(dir) Path
        +findImportFiles(dir) List~Path~
        +loadScopeConfig(dir) ScopeConfig
    }

    class ExcelService {
        +openWorkbook(path) Workbook
        +copyTemplate(src, dest) void
        +readAllCellData(path, name) List~CellData~
        +writeCellData(wb, list) void
        +saveWorkbook(wb, path) void
    }

    class ValidationService {
        +validateFileCount(files, expected) List~ValidationError~
        +validateFileNames(files) List~ValidationError~
        +validateNoDuplicates(tracker, cells) List~ValidationError~
    }

    class MergeService {
        +extractUniqueCells(import, template) List~CellData~
    }

    class CellData {
        -String coordinate
        -Object value
        -String sourceFile
        -CellValueType valueType
    }

    class ReportTask {
        -String reportName
        -String year
        -String month
        -Path templatePath
        -List~Path~ importFiles
        -Path outputDir
        -String outputFileName
        -ScopeConfig scopeConfig
    }

    class ProcessResult {
        -LocalDateTime executionTime
        -boolean success
        -int totalReports
        -int successCount
        -int failureCount
        -List~ReportResult~ reportResults
        -String summary
    }

    class ReportResult {
        -String reportName
        -String year
        -String month
        -boolean success
        -int filesProcessed
        -int cellsWritten
        -String outputFileName
        -List~ValidationError~ errors
    }

    class ValidationError {
        -ErrorType errorType
        -String message
        -String sourceFile
        -String coordinate
    }

    ReportOrchestrator --> FileService
    ReportOrchestrator --> ExcelService
    ReportOrchestrator --> ValidationService
    ReportOrchestrator --> MergeService
    ReportOrchestrator --> ReportTask
    ReportOrchestrator --> ProcessResult
    ProcessResult --> ReportResult
    ReportResult --> ValidationError
    MergeService --> CellData
    ExcelService --> CellData
    FileService --> ReportTask
    FileService --> ScopeConfig
    FileService ..> AppConfig
```

## 七、狀態機 — 單次執行生命週期

```mermaid
stateDiagram-v2
    [*] --> Initializing : 啟動
    Initializing --> ConfigLoaded : 設定載入成功
    Initializing --> Failed : 設定載入失敗

    ConfigLoaded --> Scanning : 掃描檔案
    Scanning --> Validating : 檔案掃描完成
    Scanning --> Failed : 目錄不存在

    Validating --> Processing : 檢核通過
    Validating --> Failed : 檢核失敗

    Processing --> Merging : 逐檔處理中
    Merging --> Processing : 下一份檔案
    Merging --> Failed : 重複座標

    Processing --> Outputting : 所有檔案處理完成
    Outputting --> Completed : 產出成功
    Outputting --> Failed : 寫入失敗

    Completed --> [*]
    Failed --> [*]
```
